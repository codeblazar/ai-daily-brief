# Try This Today

- **Explore Thompson Sampling:** Implement the algorithm from the [Towards Data Science article](https://towardsdatascience.com/diy-ai-ml-solving-the-multi-armed-bandit-problem-with-thompson-sampling/) to understand its mechanics in a simulated multi-armed bandit environment. - **Boost performance by calling Rust in Python:** Follow the [guide](https://towardsdatascience.com/calling-rust-from-python/) to create a simple Rust function, compile it into a Python module, and invoke it. This exercise improves your multi-language interoperability skills.
